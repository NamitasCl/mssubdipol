# Módulo: formularios

Sistema de administración/captura de formularios operativos (detalle a completar según código del módulo). Interactúa con las entidades de dominio que luego son consultadas por NODOS.

## Posibles responsabilidades
- Crear/editar formularios
- Persistencia de datos relacionados (personas, vehículos, drogas, etc.)

## Integración
- Publica datos que NODOS consulta para auditoría/estadística
